gulp的使用

npm,Node Package Manager ,node包管理工具,装了Node.js之后，npm会自动安装！

npm 命令

npm init 用来给当前项目初始化一个package.json文件
npm init -y 可以快速创建package.json 所有选项都是默认的！！

npm install 包名称@版本号
下面的命令，可以下载对应版本的包，并且将包添加到当前项目的依赖项当中（package.json文件中的dependences）
npm install 包名称@版本号 –save
npm install 包名称@版本号 -S

下面的命令，可以下载对应版本的包，并且将包添加到当前项目的开发依赖项中(package.json文件中的 devDependences)
npm install 包名称 –save-dev
npm install 包名称 -D

下面的命令会将包进行全局安装
npm install 包名称 -g

卸载已经安装的包
npm uninstall bootstrap

-S -D 都可以加载写在指令后面 用来删除package.json文件中的内容

npm官网

www.npmjs.com

cnpm 淘宝npm镜像

npm install -g cnpm –registry=https://registry.npm.taobao.org

安装了cnpm之后，就完全可以是用cnpm带来npm
cnpm和npm的唯一的不同点就在于，npm是从国外服务器下载，cnpm是从国内淘宝服务器下载！
